# Назначение

REST-сервис предоставляет API для 
- авторизации предустановленных пользователей;
- вывода списка уже загруженных файлов пользователя;
- загрузки файлов;
- изменения имени загруженного файла;
- скачивания файла.

Список предустановленных пользователей (логин/пароль):
- kes/123456
- potap/123456

Пользователи работают только со своими файлами.
После рестарта сервиса информация о загруженных файлах теряется. 

# Сборка и запуск

Для запуска сервера потребуются установленные Docker и docker-compose.

1. Выполнить в корневой директории проекта команду
``mvn package``
    
2. Запустить в корневой директории проекта команду
``docker-compose up``

После запуска сервер доступен по адресу http://localhost:8081
